# App setup

<p>Please, use json server for db.json use this command below</p>

Install JSON Server 

```
npm install -g json-server
```

Start JSON Server

```bash
json-server --watch db.json
```

Link for full doc over here 
<a href="https://github.com/typicode/json-server">

# Starting React app

fi you use npm:
```bash
npm start
```
if you use yarn:
```bash
yarn start
```
